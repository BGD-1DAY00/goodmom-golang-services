# GoodMom Backend Services

## Overview
Microservices architecture leveraging Golang and Spring Boot for the GoodMom platform.

## Services
- Golang Services: Located in `/golang-services`
- Spring Boot Services: Located in `/java-services`

## Prerequisites
- Go 1.20+
- Java 17+
- Docker
- Docker Compose
